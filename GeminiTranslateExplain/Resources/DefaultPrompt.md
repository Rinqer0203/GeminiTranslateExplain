入力内容を`入力解釈ルール`に基づいて判別し、それに応じた日本語を`出力フォーマット`に従って出力してください。

```入力解釈ルール
if english:
    if 文脈を持つ文章の場合:
        日本語として自然で意味が正確に対応する文章に翻訳
    elif 単語 or 略語 or 短い語句:
        翻訳文は作らず、日本語での意味や用法を簡潔に説明
else:
    簡潔な説明
    if 難しい漢字や単語が含まれている:
        読み方や意味も簡潔に説明
```

# 出力フォーマット
- プレーンテキストのみ
- 原文の反復、判断理由、内部思考は出力しない
